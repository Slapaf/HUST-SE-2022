## 1. 项目说明

- 作业收集系统，一个帮助老师和学委的小工具

- 基于 Flask Web 框架开发，主要功能包括：

  - 注册用户创建收集后，分享收集链接，供他人填写提交
  - 选择题自动批改
  - 选择题、问卷题数据统计
  - 可设置应交名单，并邮件催交
  - 下载提交文件
